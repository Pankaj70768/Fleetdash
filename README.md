# 🚗 FleetDash

FleetDash is a Fleet Management System developed to simplify the management of vehicles, drivers, trips, and related data. The project aims to provide an organized platform for monitoring fleet operations efficiently.

## Project Overview

The main objective of FleetDash is to digitize fleet management by storing and managing vehicle information in a centralized system. It helps reduce manual work and improves the efficiency of managing fleet-related records.

## Features

* Vehicle Management
* Driver Management
* Trip Management
* Dashboard for fleet information
* Database for storing fleet records
* Easy-to-use interface
* Scalable project structure
